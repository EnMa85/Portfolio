Il progetto gestisce un distributore di bevande tramite un'interfaccia implementata con Swing.

L'utente all'avvio può loggarsi con il proprio ID oppure attivare un nuovo account.

Ad accesso effettuato viene caricato il menù.

Il menù implementa i vari metodi per l'erogazione delle bevande, per ricaricare il credito della chiavetta e per visualizzare il saldo residuo.

Ad inizio esecuzione viene caricato il file .ser contenente i dati delle chiavette attive.

A fine esecuzione il file viene aggiornato chiamando il metodo salvaDistributore.
