Il progetto gestisce un archivio di film tramite un'interfaccia implementata con Swing ed un database MySql.

I singoli pulsanti sono implementati separatamente per mantenere pulita la gestione dell'interfaccia.

Il metodo cercaGenere esegue una query di tipo Distinct che seleziona i generi presenti nella tabella Film.
La lista ottenuta viene usata per visualizzare un elenco a tendina da cui scegliere il genere da cercare.

Una seconda query restituisce i film corrispondenti, che viene passata al metodo stampa.

La connessione con il database viene gestita da un'apposita classe.

Il metodo eseguiQuery riceve in ingresso la stringa query ed una serie di parametri di tipo Object.
Tramite un ciclo for assegna un segnaposto per ogni parametro ed esegue il prepared statement.

Se si tratta di una select, salva in result set ottenuto dal db e lo restituisce alla chiamante. Se si tratta di una variazione, restituisce una lista vuota.

Il result set ottenuto viene passato al metodo creaListaDaObject che effettua il cast ad oggetti di tipo Film.
In tal modo la classe DatabaseManager rimane aspecifica e portabile.
