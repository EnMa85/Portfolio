I miei progetti 


WEB APPS

La sezione raccoglie le applicazioni implementate con il framework Flask.

Le apps sono sviluppate in Python e HTML con elementi di CSS e JS.
Fanno uso della libreria matplotlib per la creazione di grafici e di MySql per la gestione dei dati.
PYTHON GUI

La sezione raccoglie i programmi implementati con le interfacce grafiche Tkinter.

Sono sviluppati in Python, utilizzando MySQL ed i files pickle per la gestione dei dati e la libreria matplotlib per l'elaborazione di grafici.
PYTHON CONSOLE

La sezione raccoglie i programmi Python per console.

Sono sviluppati ad oggetti e sfruttano le classi ed i principali metodi per la manipolazione di stringhe e liste.
Vengono utilizzate le librerie pickle, time, matplotlib.
