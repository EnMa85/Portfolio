Libreria

Il progetto gestisce una libreria.

La struttura si basa su due classi ed un ciclo While per il controllo delle opzioni.

La classe Libro contiene Titolo, Autore, Anno di pubblicazione e Disponibilità.

La classe Biblioteca raccoglie la lista di libri ed i metodi Stampa elenco e Prenota titolo che agiscono sui dati della classe.

L'utente, tramite console, può scegliere di aggiungere o prenotare (se disponibile) un libro, ricercare un autore, salvare o leggere la lista in un file esterno.

Per il salvataggio su file è stato usato il modulo Pickle.
